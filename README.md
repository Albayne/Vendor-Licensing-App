# Vendor-Licensing-App
A full stack mobile app that allows vendors to apply for licenses, make payments through the paynow gateway (eg. Rent) and track their financial history concerning the app. 

# 🧾 Vendor Licensing App — Backend API

A backend service designed to manage vendor licensing, registrations, renewals, authentication, and compliance workflows.  
Built for scalability, modularity, and easy integration with mobile or web front‑ends.

---

## 🚀 Features

- Vendor registration & authentication  
- License creation, renewal, and expiration tracking  
- Secure API endpoints  
- Role‑based access control  
- Database integration  
- Modular, maintainable architecture
